## Featured Cybersecurity Projects

### Enterprise Cybersecurity Capstone Lab - Blue Team / Red Team / GRC
Designed and deployed a multi-VM enterprise cybersecurity lab using virtualization, Ubuntu Server, and Kali Linux. Implemented defensive controls including SSH hardening, Fail2Ban intrusion prevention, system auditing, and control mapping to DISA STIG and NIST RMF frameworks.

### Linux Server Hardening Lab
Hardened Ubuntu Server systems by removing unnecessary services, implementing secure SSH policies, configuring authentication controls, and deploying file integrity monitoring using AIDE. Performed vulnerability assessments using Lynis.

### Blue Team Monitoring & Incident Detection
Monitored logs, open ports, and active sessions to identify suspicious activity and unauthorized access attempts. Implemented intrusion prevention rules using Fail2Ban.

### Network Reconnaissance & Intrusion Detection Exercises
Performed network discovery and service enumeration using Nmap and analyzed packet captures with Wireshark to identify suspicious traffic patterns.
